## Проект "JWT Generator"

Этот проект представляет собой пример генерации JSON Web Token (JWT) с использованием библиотеки Microsoft.IdentityModel.Tokens. JWT - это компактный формат передачи информации в виде JSON-объектов между двумя сторонами. В данном случае, проект демонстрирует создание JWT токена и сохранение его в файл.

### Инструкции по использованию

1. **Настройка конфигурации**: Отредактируйте файл `appsettings.json` для указания необходимых параметров конфигурации, таких как путь к файлу PFX с закрытым ключом, пароль для PFX, имя пользователя (ClientName) и другие, если необходимо.

2. **Зависимости**: Убедитесь, что у вас установлены библиотеки 
	- `Microsoft.IdentityModel.Tokens`
	- `Microsoft.Extensions.Configuration`
	- `Microsoft.Extensions.Configuration.Binder`
	- `Microsoft.Extensions.Configuration.Json`


3. **Код генерации токена**: Откройте файл `Program.cs` и обратите внимание на код. В этом файле содержится код, который загружает конфигурацию из `appsettings.json`, создаёт JWT токен и сохраняет его в файл `JWT.txt` на рабочем столе.
